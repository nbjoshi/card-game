## Features

- Memory-card gameplay (flip, match, reset)
- Animated UI feedback (confetti, motion)
- Fetches Fortnite cosmetic/card data (API-based)
- Responsive layout

---

## Tech Stack

- **Framework:** Next.js
- **Language:** TypeScript
- **UI:** React
- **Animation:** Framer Motion
- **Data Fetching:** Axios
- **Effects:** canvas-confetti and snow overlay
